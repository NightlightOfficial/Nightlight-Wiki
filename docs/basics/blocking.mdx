# Blocking
If a user is making you uncomfortable or you for any reason don't wish to interact with them anymore, you can block them.

### How to block someone

Blocking a user on Nightlight is simple and straightforward. There are two ways to block someone:

* Through the user's profile

1. Go on problematic user's profile
2. Click on  next to the "Message" button
3. Press "Block"

This will block the user.
* Through the user's post
* Click on  the user's post
* Press "Block \[user]"

By following the steps above, you will block the user. They will not be notified about you blocking them.

### What does blocking do

Blocking a user on Nightlight makes **you and the user** completely unable to interact with each other. Here is an incomplete list of actions that will be blocked between each other:

* Posts
* Liking
* Commenting
* Reposting
* Will not show up on the home page or in categories
* Messages
* Creating chats
* Adding to group chats
* Messaging
* This affects existing conversations as well
* Profile
* The user you blocked will not be able to visit your profile at all, instead, they will see a "404 page" which will make it seem like your profile doesn't exist.
