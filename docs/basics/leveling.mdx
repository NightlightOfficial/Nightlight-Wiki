# Leveling
You can think of levels as a sort of a trust score for users. The higher level a user is, the more time they have spent on the platform.

### What do levels allow me to do?

Having a certain level can unlock more features on Nightlight.&#x20;

* **Your level decides what size limit you have for file uploads**
* By default, everyone gets 5MB file limit when posting with media, this limit will increase overtime as you level up
* **Hourly post limit**
* Your hourly post limit will also increase as you level up, however, you can significantly increase your limit by subscribing to [Nitoa ](https://wiki.night-light.cz/basics/getting-started/broken-reference)or [Auronita](https://wiki.night-light.cz/basics/getting-started/broken-reference)
* **Group chats**
* These types of chats are new and only allow users of a certain level to participate

### How can I earn XP?

Excellent question! There are many ways to earn XP on Nightlight:

| Name                                 | Amount of XP received              |
| ------------------------------------ | ---------------------------------- |
| Nighty's Humble Reward (Daily Login) | 80 or 150                          |
| Receiving 10 or 20 likes on a post   | 50 or 100                          |
| Purchasing items using NCPs          | Dependent on the price of the item |
| Obtaining a badge                    | ¼ of required XP for level-up      |
| Referring a friend                   | 200                                |
| Selling an item on the Marketplace   | ⅕ of required XP for level-up      |
| Stellnix (Minecraft server)          | Dependent                          |
